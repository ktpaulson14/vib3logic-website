## Deployment

This website is deployed on Vercel and connected to your domain via DNS records.

### Current DNS Configuration
- **vib3logic.com**: Points to maintenance-sale hosting
- **www.vib3logic.com**: Points to Vercel deployment

### To Deploy on Vercel

1. Connect your GitHub account to Vercel
2. Import this repository
3. Set the deployment domain to your preferred hosting
4. Push updates to trigger automatic deployments
